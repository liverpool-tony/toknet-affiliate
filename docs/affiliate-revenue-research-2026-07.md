# 日本アフィリエイトマーケティング 収益ベンチマーク調査（2025-2026）

**調査日**: 2026年7月22日
**対象サイト**: AI共創レビュー研究所（toknet.info）
**現在状態**: 218記事、週26セッション、オーガニック検索0、直帰率73%、平均滞在10.3秒

---

## 1. Amazonアソシエイト コンバージョン率（商品レビューサイト）

### カテゴリ別コンバージョン率ベンチマーク

| カテゴリ | CV率（クリック→購入） | 備考 |
|---------|---------------------|------|
| **家電・PC** | 1.0〜3.0% | 高単価のため比較検討が長い。レビューサイト経由は2%前後が中央値 |
| **ガジェット・小型電子機器** | 2.0〜5.0% | 衝動買いが多く、¥5,000以下の商品で高い |
| **ソフトウェア・デジタル** | 3.0〜8.0% | 即時購入が多くCV率最高。ただしAmazon経由は限定的 |
| **ファッション** | 1.5〜4.0% | 返品率が高く実質CVは低い |
| **食品・日用品** | 5.0〜10.0% | 低単価・高頻度。ただし報酬率が低い |
| **書籍** | 3.0〜6.0% | 単価が低く報酬も小さい |

### 重要な補足
- **上記は「アフィリエイトリンククリック→購入」のCV率**
- **サイト訪問→リンククリックのCTRは別途1〜5%**（レビュー記事で2〜3%が平均）
- **実効CV率（訪問→購入）= CTR × CV率 = 0.02〜0.15%**
- AI生成レビューサイトの場合、E-E-A-T不足でCTRが通常より低い傾向（0.5〜1.5%）
- **toknet.infoの現状（直帰率73%、滞在10.3秒）では実効CV率は0.01〜0.05%と推定**

### 業界平均との比較
- 成熟した専門レビューサイト: 実効CV率 0.1〜0.3%
- 新規・低トラストサイト: 実効CV率 0.01〜0.05%
- **toknet.infoは現在「新規・低トラスト」カテゴリ**

## 3. セッション単価（Revenue Per Session）ベンチマーク

### 日本アフィリエイトサイトのRPS

| サイトタイプ | RPS（円/セッション） | 条件 |
|-------------|---------------------|------|
| **高単価ASP特化**（金融・保険） | ¥50〜200 | 1件¥5,000〜¥30,000の案件 |
| **専門レビューサイト**（家電・PC） | ¥5〜30 | Amazon 2% + ASP併用 |
| **雑記ブログ**（AdSense中心） | ¥1〜5 | AdSense RPM ¥200〜500 |
| **グルメ・旅行ブログ** | ¥2〜10 | 楽天アフィリ + AdSense |
| **AI生成レビューサイト（初期）** | **¥0〜2** | トラスト不足、CV极低 |

### toknet.infoの現状RPS推定
- 週26セッション × 実効CV率0.02% × 平均報酬¥2,000 = **週¥10.4**
- **RPS = ¥0.4/セッション**（ほぼゼロ）
- 原因: 直帰率73%、滞在10.3秒 → リンククリックすら起きていない

### 目標RPS
| 段階 | RPS | 条件 |
|------|-----|------|
| 初期（〜6ヶ月） | ¥1〜3 | 直帰率50%以下、滞在30秒以上 |
| 成長期（6〜12ヶ月） | ¥5〜15 | 検索流入あり、CV率0.1% |
| 成熟期（12ヶ月〜） | ¥15〜50 | 専門性確立、リピーターあり |

## 4. 代替・追加収益源

### 4.1 Google AdSense

| 指標 | 日本平均 | 備考 |
|------|---------|------|
| **RPM（1000PVあたり収益）** | ¥200〜500 | テクノロジー系は¥300〜600 |
| **CPC（クリック単価）** | ¥20〜80 | 金融系は¥100超も |
| **CTR（クリック率）** | 0.5〜2.0% | レビュー記事は1%前後 |

**toknet.infoでのAdSense収益推定:**
- 週26セッション × RPM ¥300 / 1000 = **週¥7.8（月¥31）**
- 500セッション/週なら: **週¥150（月¥600）**
- 5,000セッション/週なら: **週¥1,500（月¥6,000）**

**注意点:**
- 218記事あってもトラフィックがなければAdSense収益はほぼゼロ
- AI生成コンテンツはAdSenseポリシー上「価値の低いコンテンツ」に該当するリスク
- 2024年以降、GoogleはAI生成コンテンツのAdSense審査を厳格化

### 4.2 楽天アフィリエイト

| 指標 | 数値 |
|------|------|
| 報酬率 | 2〜8%（カテゴリ別） |
| 平均単価 | ¥3,000〜5,000 |
| CV率 | Amazonよりやや高い（3〜5%） |
| クッキー有効期間 | 30日（Amazonは24時間） |

**メリット:** クッキー期間が長く、CV率が高い
**デメリット:** 報酬率がAmazonより低い場合が多い、商品データが古い

### 4.3 自社商品・デジタルコンテンツ

| 収益源 | 月額ポテンシャル | 難易度 |
|--------|----------------|--------|

| **Brain/Tipstower** | ¥10,000〜100,000 | 高（実績が必要） |
| **有料メルマガ** | ¥10,000〜200,000 | 高（読者基盤が必要） |

**相性:** 商品レビュー動画 → ブログ詳細記事 → Amazonリンクの導線は有効。ただし動画制作コストが高い。

## 5. 成功している日本アフィリエイトサイトの事例と戦略

### 5.1 成功サイトの共通パターン

#### パターンA: 専門特化型（月収50万〜500万円）
- **例**: 家電批評系ブログ、PCパーツ専門サイト
- **戦略**: 1カテゴリに絞り、実機レビュー + 比較表 + 動画
- **トラフィック**: 月10万〜100万PV
- **収益源**: Amazon + ASP + メーカー案件
- **成功要因**: E-E-A-Tの「Experience」が圧倒的。実機写真・ベンチマークデータ

#### パターンB: 比較・ランキング型（月収10万〜100万円）
- **例**: 「おすすめ〇〇ランキング」系
- **戦略**: SEOで「〇〇 おすすめ」キーワードを大量取得
- **トラフィック**: 月5万〜50万PV
- **収益源**: ASP高単価案件 + Amazon
- **成功要因**: 網羅性 + 更新頻度。ただし2024年以降はAI生成で飽和

#### パターンC: SNS流入型（月収5万〜50万円）
- **例**: X（Twitter）やInstagramで商品紹介
- **戦略**: SNSでバズ → ブログで詳細 → Amazon/楽天
- **トラフィック**: SNS依存で不安定
- **収益源**: Amazon + 楽天 + PR案件
- **成功要因**: フォロワーとの信頼関係、キャラ立ち

#### パターンD: AI活用型（新興、月収1万〜30万円）
- **例**: AI生成レビュー + 人間編集のハイブリッド
- **戦略**: AIで大量記事 → 人間が体験・専門性を追加
- **トラフィック**: SEOは不安定、SNSで補完
- **収益源**: Amazon + AdSense
- **成功要因**: 透明性（AI使用を明記）+ ニッチ特化 + 人間の付加価値

### 5.2 toknet.infoが「やっていない」こと（成功サイトとの差）

| 成功サイトの施策 | toknet.infoの現状 | 優先度 |
|----------------|-----------------|--------|
| 実機写真・動画 | AI生成テキストのみ | 🔴 高 |
| 比較表・スペック表 | なし or 不十分 | 🔴 高 |
| 内部リンク構造 | 不十分 | 🔴 高 |
| ピラーページ（選び方ガイド） | 不足 | 🟡 中 |
| SNSでの積極発信 | Mastodon自動投稿のみ | 🟡 中 |
| 人間の実体験・レビュー | AI生成のみ | 🔴 高 |
| 構造化データ（レビューSchema） | 未確認 | 🟡 中 |
| 読者コミュニティ | なし | 🟢 低 |
| メールマガジン | なし | 🟢 低 |

### 5.3 2025-2026年の市場環境変化

1. **Google Helpful Content Update（HCU）の継続強化**: AI生成のみのコンテンツはインデックスすらされない
2. **AI Overview（SGE）の日本展開**: 検索結果でAIが直接回答 → クリック率低下
3. **ステマ規制（2023年10月〜）**: PR表記必須、違反で措置命令
4. **Amazon AI表記義務（2026年4月〜）**: AI生成コンテンツの明記必須
5. **アフィリエイト市場の成熟**: 参入者増加で競争激化、差別化が必須

## 7. 総合評価と提言

### このビジネスモデルは成立するか？

**結論: 現状のAI生成レビューのみでは成立しない。ピボットが必要。**

#### 成立しない理由
1. **週26セッションでは物理的に収益が出ない**（月¥20〜40）
2. **オーガニック検索ゼロ** = Googleにインデックスされていない or 評価されていない
3. **直帰率73%・滞在10.3秒** = コンテンツが読まれていない
4. **AI生成のみ** = E-E-A-Tの「Experience」がゼロ、Google HCUで不利
5. **PC・家電カテゴリは報酬率2%** = 高トラフィックが必要

#### 成立させるための条件
1. **最低でも週500セッション**（月間収益¥5,000〜10,000の閾値）
2. **実効CV率0.1%以上**（リンクCTR 2% × Amazon CV 5%）
3. **収益源の多様化**（Amazon一本足は危険）
4. **人間の付加価値**（実体験、専門知識、独自データ）

### 優先アクション（ROI順）

| 優先度 | アクション | 期待効果 | 工数 |
|--------|-----------|---------|------|
| **P0** | Google Search Consoleでインデックス状況確認・修正 | 検索流入の前提条件 | 1日 |
| **P0** | 直帰率改善（記事冒頭に結論、目次、比較表追加） | 滞在時間↑、CTR↑ | 1週間 |
| **P0** | 内部リンク構造の構築（ピラー↔クラスター） | SEO評価↑ | 1週間 |
| **P1** | 高報酬カテゴリへの特化（スマートホーム4%、スポーツ4%） | RPS↑ | 2週間 |
| **P1** | 楽天アフィリエイト・もしもアフィリエイト追加 | 収益源多様化 | 3日 |
| **P1** | SNS（X）での手動発信強化 | 直接流入↑ | 継続 |
| **P2** | 実機レビュー記事の追加（月2〜4本） | E-E-A-T↑ | 継続 |
| **P2** | AdSense申請（コンテンツ品質改善後） | 副収入 | 1日 |
| **P3** | note/Brainで「AI共創レビューの作り方」販売 | 高単価収益 | 1ヶ月 |
